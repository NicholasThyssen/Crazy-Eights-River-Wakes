# Crazy-Eights-River-Wakes

A Virtual Reality Experience were players sit on a raft and play Crazy Eights while enjoying the nature themed decks along with the nature around in the scenery.

(later will have picture/ GIF here)

## Requirements

* Unity Hub
* Unity 6000.0.67f1
* Git

## Getting Started

1. Clone the Repository

```bash
git clone https://github.com/NicholasThyssen/Crazy-Eights-River-Wakes
cd Crazy-Eights-River-Wakes
```

2. Open It On Unity Hub

On Unity Hub click add project from disk and choose the folder of the whole repository.

3. Click on game to launch and you can start editing it.

Now you can add any changes you would want to be implemented into the game!

## Comitting

Do not forget to commit your finished working and to pull any changes to minimize merge conflicts. Will make different branches for some of the developments. Make sure never to work on another scene someone else is currently working or making changes too otherwise will have very bad merge conflicts.
